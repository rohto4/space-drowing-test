# ロール: シニア・クリエイティブデベロッパー (WebGL/Three.js エキスパート)
# 依頼: タスク管理ツール「Chronos Rewards」の宇宙観測ビュー（プロトタイプ）の実装

## 開発の目的
「自分の生きた時間が宇宙になる」というコンセプトを検証するため、非常に高品質でシネマティックな銀河描画コンポーネントを作成してください。
「デフォルトのThree.js感」を排除し、ポストプロセッシングと数理ロジックを駆使して「高級感」と「奥行き」を表現することが最優先事項です。

## 技術スタック
- React (Vite) / Tailwind CSS
- Three.js / React Three Fiber (R3F)
- @react-three/drei / @react-three/postprocessing
- GSAP (カメラワーク制御)

## 実装すべき視覚的要件（フェーズ演出）

### 1. 8bit・スターウォープ (導入)
- 背景は漆黒 (#000000)。
- 中央から放射状に広がる星々を「正方形のピクセル」として描画してください。
- 時空を移動しているような加速感のあるアニメーションを実装してください。
- 画面全体にピクセル化フィルタ（Pixelation）をかけ、レトロデジタルな質感を演出してください。

### 2. 高精細銀河への進化 (遷移)
- ボタンクリック等のトリガーで、ピクセルフィルタを解除し、高精細3D空間へシームレスに遷移させてください。
- 四角いドットが「光り輝く粒子」へと融解し、UnrealBloomによる美しい滲みが発生する演出を行ってください。

### 3. プロシージャル銀河の構築
- 対数螺旋（Logarithmic Spiral）の数式を用いて、数千個の星を銀河状に配置してください。
- パフォーマンス向上のため `InstancedMesh` を使用してください。
- 星の色はジャンル（仕事:青、学習:黄、健康:赤など）に基づいたパレットを割り当ててください。
- 物理ベースレンダリング (PBR) を適用し、各星が自発光しているような質感を `MeshStandardMaterial` や `Emissive` で設定してください。

### 4. シネマティック・ポストプロセッシング
- UnrealBloomPass: 強い光だけが霧の中に滲むような設定。
- Fog (指数関数的霧): 宇宙の奥に行くほど濃紺の霧に消える演出。
- ACESFilmicToneMapping: 映画のような色調補正。
- Noise / Vignette: デジタル特有の「平坦さ」を消すためのノイズと四隅の減光。

### 5. カメラワーク
- GSAPを使用し、最初は一つの星にクローズアップした状態から、一気に「引き（マクロ視点）」までカメラを引く、ダイナミックなオープニングを実装してください。

## ダミーデータ
- 過去数ヶ月分の完了タスク（ジャンル、日時、重要度）を500件ほどシミュレートして銀河に反映してください。

## 出力
- これらを一つのReactコンポーネント、または整理されたファイル群として出力してください。
- シェーダーコードやポストプロセッシングのパラメータ調整に特に注力し、「圧倒的な没入感」をコードで表現してください。
